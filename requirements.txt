# =============================================
# PASSO A PASSO PARA RODAR EM OUTRA MÁQUINA
# =============================================
#
# 1. Clone o repositório:
#    git clone https://github.com/jpfbarri/SBF-Dashboard.git
#
# 2. Abra o terminal na pasta do projeto:
#    cd SBF-Dashboard
#
# 3. Crie o ambiente virtual (só precisa fazer 1 vez):
#    python -m venv .venv
#
# 4. Ative o ambiente virtual:
#    Windows (PowerShell): & .venv\Scripts\Activate.ps1
#    Windows (CMD):        .venv\Scripts\activate.bat
#    Linux/Mac:            source .venv/bin/activate
#
# 5. Instale as dependências (só precisa fazer 1 vez):
#    pip install -r requirements.txt
#
# 6. Rode o dashboard:
#    streamlit run dashboard.py
#
# 7. Acesse no navegador:
#    http://localhost:8501
#
# 8. Para parar o dashboard:
#    Pressione Ctrl + C no terminal
# =============================================
